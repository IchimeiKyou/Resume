長期インターンシップ

  • 国金証券株式会社　エンジニア（LLM評価・アプリケーション開発）（2025年3月～2025年6月）
    • LLM評価：Agentタスクを対象に、Human/LLM-as-a-Judge方式を融合した「手作業作成＋LLM拡張」による40以上の基礎サブタスクを構築。ファイル操作、コード生成、可視化、ツール呼び出しなどの組み合わせで問題難易度（Level 1–4）を定義。ユーザー指示理解、ツール呼び出し精度、タスク達成度の3軸から加重戦略（0.3/0.3/0.4）を設定し、Claude-3.5-Sonnet、GPT-4.1、DeepSeek-R1等を評価。GPT-4.1が総合最優（Level1–4得点：0.963/0.931/0.900/0.856）で、Agent基盤モデルに選定。
    • アプリケーション開発：MCPベースのリサーチレポート再現Agentシステムを構築。LLM、Data MCP Server（知識庫検索・テキスト解析・Text-to-SQLの統合）、Instructor MCP Server（指示駆動型の再現プロセス、戦略タイミング・マルチファクター株選択対応）の3コンポーネントにより、レポート取得、データ準備、モデル構築、コード生成、レポート出力の全工程を自動化。また、異常検知モジュールを実現し、モデル自己検証、バイパスレビュー、ユーザー介入で全工程の正確性を担保。6月リリース後、9月までに百件以上の課題委託を完了、社内主要製品として顧客向け展開。

自己PR

私の強みは、新しい技術の習得力と問題解決力、そしてチームでの調整力です。
国金証券株式会社でのインターンシップでは、リサーチレポート再現Agentシステムの開発を担当しました。当初、MCPに関する知識はほとんどなく理解が難しい状況でしたが、既存資料を調べ、担当者の方に学ぶことで概念や原理を習得し、データMCP Serverと指令MCP Serverを自ら構築しました。システム全体の開発過程では、業務要件の変更や期待通りの成果が得られない課題に直面しましたが、システムフロー図を作成し担当者と丁寧にコミュニケーションを取り、プロンプト調整や自動エラー検知モジュールを導入するなど試行錯誤を重ね、安定稼働するシステムを完成させることができました。この経験により、新しい技術を素早く学び、問題解決に応用する力を実感しました。
また、東方証券株式会社での企業価値評価モデル構築プロジェクトでは、正職担当者とインターンチームの調整を行い、チーム内でデータ収集やモデル設計の手法に関して意見が分かれる課題に直面しました。私は各メンバーの意見を丁寧に聞き取り、議論をマインドマップで可視化し利点・リスクを整理したうえで、追加の小規模データを用いた試行錯誤を実施。最終的に実験結果に基づき最も実行可能な手法を決定し、期日内に複数モデルによる企業価値予測を完成させ、分析担当者からも高い評価を得ました。この経験から、チームで協働する際の調整力と効率的な意思決定の重要性を学びました。
これらの経験を通じて培った学習力、問題解決力、チーム調整力を活かし、貴社のプロジェクトにおいて技術を通じた価値創出に貢献したいと考えています。
